🎮 ARENA: ADAPT

A fast-paced survival arena game built using Phaser.js, where enemies dynamically chase and pressure the player.

🚀 Features
🧠 Enemy AI with real-time chasing behavior
⚔️ Combat system with attack mechanics
❤️ Health system with damage feedback
🎯 Score tracking and enemy respawn
🏆 High score system using localStorage
🎮 Smooth keyboard controls
📺 Multiple scenes (Loading, Menu, Game, Game Over)
🎮 Controls
Action	Key
Move	Arrow Keys
Attack	Space
🛠 Tech Stack
JavaScript (ES6)
Phaser.js
HTML5 Canvas
▶️ How to Run
Clone the repository:
git clone https://github.com/utkarsh12377/arena-adapt.git
Open in VS Code
Run using Live Server
