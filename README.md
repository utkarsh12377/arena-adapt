# 🎮 ARENA: ADAPT

A fast-paced **survival arena game** built using **Phaser.js**, where enemies dynamically chase and pressure the player in real time.

---

## 🚀 Features

* 🧠 Adaptive enemy AI (real-time chasing)
* ⚔️ Combat system with attack mechanics
* ❤️ Health system with visual feedback
* 🎯 Score tracking with enemy respawn
* 🏆 High score persistence (localStorage)
* 🎮 Smooth keyboard controls
* 📺 Multiple scenes (Loading, Menu, Game, Game Over)

---

## 🎮 Controls

| Action | Key        |
| ------ | ---------- |
| Move   | Arrow Keys |
| Attack | Space      |

---

## 🛠 Tech Stack

* JavaScript (ES6)
* Phaser.js
* HTML5 Canvas

---

## ▶️ How to Run

```bash
git clone https://github.com/utkarsh12377/arena-adapt.git
```

* Open in VS Code
* Run using **Live Server**
